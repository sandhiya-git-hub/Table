# Table Generator

## Description
This project is a C program that generates a P × P table containing random even numbers between 2 and 16. The user can choose an even number to highlight, and the program displays the updated table along with the number of occurrences of the selected value.

## Features
- Accepts the array size (P × P) from the user.
- Generates random even numbers between 2 and 16.
- Displays the generated table.
- Highlights the selected even number.
- Counts and displays the occurrences of the selected number.
- Handles invalid and out-of-range inputs.

## Language
C

## How to Compile

```bash
gcc Table.c -o Table
```

## How to Run

```bash
./Table
```

(On Windows)

```bash
Table.exe
```
